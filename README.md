# 📻 MP3.com Infinite Radio

A free, open, forever-random music station powered by the [MP3.com Rescue Barge](https://archive.org/details/mp3-com-rescue-barge) collection on Internet Archive.

430,000+ forgotten songs from the early internet. One play button. No algorithms. No ads.

**Live site:** https://benbjammin84-dot.github.io/mp3com-radio

---

## How it works

- `tracks.json` is a catalog of all MP3 URLs + metadata from the IA collection.
- The front-end loads the catalog once, keeps a random shuffle queue in memory, and streams each track directly from Internet Archive.
- Your browser only ever touches 1-2 tracks at a time. No downloads. No lag.

---

## Building the catalog

The catalog (`tracks.json`) is generated by `build_catalog.py`.

```bash
pip install requests
python build_catalog.py
```

This will take a while (39 items, ~430k tracks). Run it once, commit `tracks.json`, done.

---

## Stack

- Pure HTML / CSS / JS front-end
- Static hosting via GitHub Pages
- All audio streamed from Internet Archive
- No backend. No database. No cost.
